# dev_et08a

## 概述
概述
`dev_et08a`是一个用于处理et08a遥控器数据的驱动模块，支持通过DMA接收数据并解析遥控器的输入。

## 遥控器配置
et08a遥控器与dr16最大的不同就是需要自行修改遥控器本身设置，请根据以下格式进行配置：
- 系统设置
  - 教练：教练模式，全部通道都选择开启
  - 摇杆模式：模式2 
- 通信设置
  - 对码：模式A
  - PPM/W.BUS: W.BUS
  - 接力飞行：禁用
  - 
- 通用功能
  - 通道设置按顺序为(不关心微调)：J1,J2,J3,J4,SA,SB,SC,SD
  - 正反设置按顺序为：正,反,反,正,正,正,正,正
  - 辅助微调：全0
  - 通道延时：禁用
  
## 注意事项
1. 请将对应Uart的DMA配置成循环模式，波特率100000，停止位1位，偶校验
2. 在发现结算出来的数据与遥控器的实际通道不一致的话，清更具`遥控器配置`章节重新配置遥控器
3. 如不可避免的还是发生数据错位，请进行reset操作
